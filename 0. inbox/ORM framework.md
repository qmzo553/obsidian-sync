#### ORM Framework 란?
>ORM(Object-Relational-Mapping)은 데이터베이스와 객체 지향 언어 간의 호환되지 않는 데이터를 변환하는 프로그래밍 기법이다.
> 객체 관계 매핑이라고도 한다.
