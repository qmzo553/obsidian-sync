### 메모
> Git의 역사
> 	- 리눅스 커널은 패치 파일과 단순 압축 파일로 관리
> 	- 2002년 BitKeeper라는 상용 DVCS를 사용
> 	- 2005년에 BitKeeper의 무료 사용이 제고됨 -> 리눅스 개발 커뮤니티가 자체 도구를 만드는 계기
> 	- 2005년에 Git 탄생

> 델타가 아니라 스냅샷
> 	Subversion이나 각 파일의 변화를 시간순으로 관리
