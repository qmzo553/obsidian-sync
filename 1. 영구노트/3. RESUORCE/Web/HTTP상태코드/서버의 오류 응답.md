### 주제 : 서버 오류응답 (5xx)

### 날짜 : 2024-05-10 08:07
----
### 메모
> 서버 오류응답
> 	- 서버가 유효한 클라이언트 요청을 처리할 수 없음을 나타낸다.
> 	- 서버 내부의 문제로 인해 발생하며, 문제가 서버 측에 있음을 클라이언트에게 알리는 데 사용된다.
> 
> 상태 코드 설명
> 	- 500 Internal Server Error : 가장 일반적인 서버 오류, 서버가 요청을 처리하는 중 예기치 않은 상황이 발생했을 때 사용
> 	- 501
