### 메모
> JSTL(JSP Standard Tag Libraray)
> 
> core tag library
> 일반
> 	- <c:out> = <%= %>
> 	- <c:set>
> 	- 변수 생성, 변수 값 할당
> 	- JSP 로컬 변수가 아닌 지정한 scope의 저장소에 저장됨
> 	- <c:remove> : 변수 삭제
> 
> 조건문 / 반복문
> 	- <c:if>
> 	- <c:choose>
> 	- <c:forEach>
> 
> 기타
> 	- <c:url>
> 	- <c:param>
> 	- <c:redirect>
> 
> function tag library
> 	- 함수 사용 가능
> 
> fmt
> 	- 다국어 메세지 처리
> 
> 예외 처리
> 	- 에러 처리용 페이지
> 	- JSP 내장 객체 exception 사용
