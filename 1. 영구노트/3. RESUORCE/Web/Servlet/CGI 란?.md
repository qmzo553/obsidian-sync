> 	- Common Gateway Interface
> 	- 웹 서버가 외부 프로그램을 실행할 수 있도록 해주는 인터페이스 명세(specification)
> 	- 외부 프로그램은 동적 웹 컨텐츠를 생성하는 역할이다. (c, c++, java 등)
> 	- 웹 서버와 CGI 프로그램 간의 규칙
