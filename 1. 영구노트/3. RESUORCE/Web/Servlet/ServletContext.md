### 메모
> Context란?
> 	- 사전적 의미 : 문맥, 상황, 배경
> 	- 소프트웨어 개발에서는 Execution Context의 의미로 많이 사용
> 	- 실행 환경 그 자체 (runtime)
> 	- 설정 정보(Configuration)

> ServletContext란?
> 	- Servlet Container 실행 환경
> 	- Servlet과 Servlet Container 간에 연
